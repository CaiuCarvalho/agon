# Product Overview

**Agon** is a premium Brazilian e-commerce platform specializing in sports merchandise, particularly Brazilian national team products and football-related items.

## Core Features

- **Storefront**: Product catalog with search, filters, and detailed product pages
- **Authentication**: Email/password auth via Supabase with user profiles and avatar uploads
- **Shopping**: Persistent cart and wishlist (max 50 items) with real-time sync
- **Checkout**: Complete flow with shipping forms and Mercado Pago integration (credit card, PIX, boleto)
- **Admin Panel**: Product CRUD, order management, and dashboard analytics
- **Payments**: Webhook-based payment processing with automatic order status updates

## Target Market

Brazilian consumers interested in national team merchandise and football products. UI/UX emphasizes Brazilian national colors (green, yellow, blue) and regional testimonials.

## Business Model

Direct-to-consumer e-commerce with integrated payment processing through Mercado Pago (Brazil's leading payment gateway).

## Key User Flows

1. Browse products → Add to cart → Checkout → Payment → Order confirmation
2. Create account → Manage profile → Track orders
3. Admin: Manage products → View orders → Update inventory
