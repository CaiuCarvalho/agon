# Como Adicionar a Imagem da Copa do Mundo

## ✅ Código Já Está Pronto!

O código já foi atualizado para usar a imagem da Copa do Mundo FIFA. Agora você só precisa salvar a imagem no local correto.

## 📁 Onde Salvar a Imagem

**Caminho completo**: `apps/web/public/ui/world-cup-trophy.jpg`

## 🔧 Passos

### Opção 1: Via Explorador de Arquivos (Mais Fácil)

1. Abra o explorador de arquivos do Windows
2. Navegue até a pasta do projeto: `agon/apps/web/public/ui/`
3. Salve a imagem da Copa do Mundo como `world-cup-trophy.jpg` nessa pasta
4. Pronto! A imagem aparecerá automaticamente no site

### Opção 2: Via Terminal

```bash
# Navegue até a pasta do projeto
cd agon/apps/web/public/ui/

# Copie a imagem para essa pasta e renomeie para world-cup-trophy.jpg
# (use o comando copy do Windows ou arraste e solte)
```

## 🎯 Resultado

Após salvar a imagem, ela aparecerá na seção **"Estrutura de um Campeão"** da homepage, mostrando o troféu dourado da Copa do Mundo FIFA.

## ✅ Verificação

1. Salve a imagem no local correto
2. Recarregue a página no navegador (F5)
3. A imagem deve aparecer na seção "Estrutura de um Campeão"

## 🐛 Se a Imagem Não Aparecer

1. Verifique se o nome do arquivo está correto: `world-cup-trophy.jpg` (sem espaços, tudo minúsculo)
2. Verifique se está na pasta correta: `apps/web/public/ui/`
3. Recarregue a página com Ctrl+F5 (limpa o cache)
4. Se ainda não funcionar, reinicie o servidor de desenvolvimento
