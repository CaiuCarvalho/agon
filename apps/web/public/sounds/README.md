# Notification Sound

## Required File

Place a notification sound file at:
```
apps/web/public/sounds/notification.mp3
```

## Requirements

- **Format**: MP3
- **Duration**: 1-2 seconds
- **Volume**: Pleasant, not jarring
- **Size**: < 100KB recommended

## Suggested Sources

1. **Free Sound Libraries**:
   - [Freesound.org](https://freesound.org/)
   - [Zapsplat](https://www.zapsplat.com/)
   - [Notification Sounds](https://notificationsounds.com/)

2. **Recommended Search Terms**:
   - "notification bell"
   - "success chime"
   - "order alert"
   - "positive notification"

## Example Sound

For testing, you can use a simple notification sound like:
- A short bell chime
- A gentle "ding" sound
- A subtle success tone
