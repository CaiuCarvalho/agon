# Skills Operacionais de Agentes (Agon)

Este diretório contém skills prescritivas para agentes de execução, com foco em:
- continuidade entre sessões;
- aderência estrita ao SDD;
- redução de erro em agentes menos capazes.

## Ordem recomendada de leitura
1. `sdd-orchestrator.md`
2. `architecture-guardrails.md`
3. `tech-stack-rules.md`
4. skill específica da tarefa (`testing-quality.md`, `ui-design-system.md`, `bugfix-regression.md`)
5. `handoff-notion.md`

## Skills disponíveis
- `sdd-orchestrator.md`: fluxo obrigatório de Spec-Driven Development com gates.
- `architecture-guardrails.md`: limites de arquitetura e colocação de código.
- `tech-stack-rules.md`: regras por tecnologia e anti-patterns.
- `testing-quality.md`: estratégia de testes e critérios de aceite.
- `ui-design-system.md`: regras para UI/UX e consistência visual.
- `bugfix-regression.md`: protocolo para correção segura de bugs.
- `handoff-notion.md`: padrão de atualização de estado e handoff no Notion.

## Regra de ouro
Sem spec válida e sem checklist de verificação, a implementação não inicia.
